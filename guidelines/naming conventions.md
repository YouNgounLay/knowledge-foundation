
**Naming a Folder**: the name of a newly created folder should be in all lowercases unless it's referring to a concept, in such case *title* format is allowed. All folders should be named with general/abstract concept in mind.

**Adding Tag**: when adding a new tag, they should all be followed a *title* format, with each tag should only covers one word. If a tag is best described with multiple words, separate them, otherwise use at most two words.
